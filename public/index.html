Technologies Covered

Python for Data Analytics | Tool: Jupyter Notebook / VS Code | 30–40 hours

  • Python Basics: Syntax, Variables, Loops, Conditions
  • Functions, Modules, and File Handling
  • Data Structures: List, Tuple, Dictionary, Set
  • Numpy for Numerical Computation
  • Pandas for Data Cleaning and Manipulation
  • Data Visualization (Matplotlib, Seaborn)
  • Working with APIs & JSON
  • Regular Expressions for Text Processing
  • Final Project: Data Analysis Use Case

SQL (Structured Query Language) | Tool: MySQL | 20–25 hours

  • Introduction to Databases and RDBMS
  • DDL, DML, DQL, and DCL Commands
  • Filtering, Grouping, Aggregating (WHERE, GROUP BY, HAVING)
  • Joins (Inner, Left, Cross, Right, Full, Self)
  • Subqueries & Nested Queries
  • CTEs and Window Functions (RANK(), ROW_NUMBER(), etc.)
  • Case Statements & Conditional Logic
  • Views, Indexes, Transactions
  • Stored Procedures (Basics)
  • Real-Time Industry Case Studies

Power BI | Tool: Power BI Desktop | 20–25 hours

  • Power BI Interface and Data Loading
  • Power Query Editor for Data Transformation
  • Data Modeling and Relationships
  • Calculated Columns, Measures using DAX
  • Time Intelligence Functions
  • Interactive Dashboards & Drillthroughs
  • Filters, Slicers, Bookmarks
  • KPI, Cards, and Custom Visuals
  • Report Publishing & Sharing on Power BI Service
  • Project: End-to-End BI Dashboard

Microsoft Excel for Analytics | 15–20 hours

  • Excel Basics and Data Cleaning
  • Functions: VLOOKUP, HLOOKUP, INDEX-MATCH, IF, COUNTIF, etc.
  • Conditional Formatting & Data Validation
  • Pivot Tables and Charts
  • Dashboards with Slicers and Dynamic Ranges
  • Scenario Analysis, Goal Seek, Solver
  • Excel + Power BI Integration

PySpark (Apache Spark using Python) Tool: Databricks | 25–30 hours

  • Introduction to Big Data & Spark Ecosystem
  • RDDs vs DataFrames vs Datasets
  • Reading Data from CSV, JSON, Parquet
  • Transformations and Actions
  • Spark SQL and DataFrame Operations
  • Writing ETL Pipelines in PySpark
  • Handling Nulls, Joins, Aggregations
  • Partitioning, Caching, Optimization
  • Working with UDFs and Window Functions
  • Project: ETL Pipeline Development

AWS for Data | AWS Console / CLI | 20–25 hours

  • Introduction to AWS Cloud Computing
  • IAM, EC2, S3 – Storage & Compute Basics
  • AWS Glue: Crawlers, Jobs, Catalogs
  • Redshift Basics and Athena Queries
  • Lambda Functions (Intro)
  • Setting up EMR Clusters
  • CloudWatch, Logging, Monitoring
  • Real-life Project: Data Pipeline with S3 + Glue + Redshift

Git & GitHub (Version Control) | Git CLI, GitHub | 6–8 hours

  • Git Installation and Configuration
  • Initializing a Repository
  • Git Basics: Add, Commit, Push, Pull
  • Branching & Merging, Resolving Conflicts
  • Cloning & Forking Repositories
  • GitHub PRs and Collaboration
  • Best Practices for Version Control
  • Git with Python/Jupyter Projects

Jira (Agile Project Management) | Tool: Jira Software

  • Introduction to Agile & Scrum Methodologies
  • Jira Interface Overview
  • Creating Epics, Stories, Tasks, Bugs
  • Sprint Planning and Execution
  • Managing Backlogs and Boards
  • User Roles and Access Control
  • Reporting and Dashboard Features
  • Real-Time Case Study: Agile Workflow Simulation

Linux for Data Professionals | 8–10 hours

  • Linux File System & Navigation
  • Basic Shell Commands (ls, cd, mkdir, rm, cp, etc.)
  • File Permissions and User Management
  • Process Monitoring (top, ps, kill)
  • Working with Files: grep, awk, sed, head, tail
  • Bash Scripting Basics
  • Package Installation & System Updates
  • Using Linux for Python, Git, and AWS CLI
